
## Installation

cd into directory

```bash
npm i
```

## Run UI

```bash
npm run start
```

## Run Server
Separate terminal
```bash
npm run server
```
## Run Tests

```bash
npm test
```

## Usage

Pass in url address for github repo into input field <br/>
Example: https://github.com/dderojas/nodejs-coding-challenge <br/>
Note: url works with our without https:// <br/>
Press button for results in console.
